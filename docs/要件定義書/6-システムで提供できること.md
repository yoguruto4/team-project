#  システムで提供すること

- 申請者が簡単に使える申請作成・提出機能

    - 交通費入力テンプレート
        - よく使う乗車区間をテンプレートとして保存・再利用可能
        - テンプレート選択で入力工数を削減

    - 直感的なUI
        - PC・スマホ両方に対応したレスポンシブデザイン
        - フォーマットに沿って順に入力できる構成

- 承認・確認作業を効率化する機能

    - 承認・保留・却下などのステータス管理
        - 一覧画面で処理状況を即座に把握可能
        - 絞り込み・検索で過去の申請もすぐに確認

    - 通知機能
        - 承認待ち・却下などのステータスに応じて通知


- 管理者による申請管理の利便性向上

    - 申請データの一括管理機能
        - ユーザーや部署単位でのフィルタリング機能
